How to deploy this system

The admin trains students here and exports the training data. You then place both files on any static web host (GitHub Pages, Netlify, etc). Students only ever open index.html — their face is checked against the exported data automatically.

Downloads the trained face descriptors for all 0 student(s).

1
Register all students using the panel on the left. Upload a clear face photo for each student with their Roll No, Branch and Year.
2
Click "Download students.json" above. This file contains the trained face data — no photos, just math.
3
Host both files on GitHub Pages, Netlify, or any static host — put admin.html, index.html and students.json in the same folder.
4
Students open index.html (your main link). Their face is detected automatically and attendance is marked.
You open admin.html to manage students and see the log.
5
When you add more students, re-export students.json and replace the file on your host. No re-deploy needed — just replace the JSON.
